# Frontend

Volto {{ cookiecutter.volto_version }} frontend for {{ cookiecutter.title }}.

Volto addon: `{{ cookiecutter.__volto_addon_name }}`

## Setup

```bash
make install    # pnpm install + mrs-developer checkout
make run        # start Volto dev server on http://localhost:3000
```

Requires Node.js {{ cookiecutter.node_version }} and pnpm {{ cookiecutter.pnpm_version }}.

## Development

| Command | Description |
|---------|-------------|
| `make install` | Install (pnpm + mrs-developer) |
| `make run` | Start Volto dev server |
| `make volto-build` | Production build |
| `make test` | Run Jest unit tests |
| `make check` | Run ESLint + Prettier + Stylelint |
| `make format` | Auto-format code |
| `make i18n` | Sync translations |
| `make volto-storybook-start` | Start Storybook |
| `make volto-storybook-build` | Build static Storybook |
| `make clean` | Remove node_modules and build artifacts |

## Build System

The `Makefile` is generated by [mxmake](https://github.com/mxstack/mxmake)
with the `volto` topic. It manages pnpm dependencies, mrs-developer source
checkouts, and QA tools.

To regenerate after changing `mxmake-preseed.yaml`:

```bash
mxmake init --preseed mxmake-preseed.yaml
```

## Addon Development

The project's Volto addon lives in `packages/{{ cookiecutter.__volto_addon_name }}/`.
This is where you add components, blocks, views, and other Volto customizations.

Translations are managed via `make i18n`, which syncs `.po` files in the
addon's `locales/` directory.
